## Getting Started

다음의 간단한 작업 후에 프로그램을 실행시킬 수 있습니다.

### Installation

1. 이 레파지토리를 Clone합니다.
```
git clone https://github.com/sangeun99/notepad.git
```

2. python 패키지를 설치합니다.
```
pip install -r requirements.txt
```

3. 메인 파일을 실행합니다.
```
python app.py
```
혹은
```
flask run
```
